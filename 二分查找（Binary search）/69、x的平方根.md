# 69、x的平方根

- 题目要求：实现 int sqrt(int x) 函数。计算并返回 x 的平方根，其中 x 是非负整数。由于返回类型是整数，结果只保留整数的部分，小数部分将被舍去。

- 示例1：

  >```
  >输入: 4
  >输出: 2
  >```

- 示例2：

  >```
  >输入: 8
  >输出: 2
  >说明: 8 的平方根是 2.82842..., 
  >     由于返回类型是整数，小数部分将被舍去。
  >```

## 方法一：二分查找  100%

```java
class Solution {
    public int mySqrt(int x) {
        int lk = 0, rk = x, ans = -1;
        int mid;
        while(lk <= rk){
            mid = (lk + rk) / 2;
            if((long)mid * mid < x){
                lk = mid + 1;
            }else if((long)mid * mid > x){
                rk = mid - 1;
            }else{
                return mid;
            }
        }
        return rk;
    }
}
```

- 时间复杂度：*O*(log（n））二分查找时间复杂度
- 空间复杂度：O(1）没有用到额外空间
- **思路**：基础二分查找思想。在二分查找的每一步比较mid平方和x大小，之后根据比较结果更新左右边界。最后返回rk
